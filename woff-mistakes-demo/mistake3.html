間違い3: 認証後も再度ログイン処理を呼ぼうとする

❌ この実装は無駄です！

// OAuth認証後のリダイレクト時
woff.init({ woffId }).then(() => {
    if (!woff.isLoggedIn()) {
        woff.login(); // 不要！URLパラメータがあれば自動認証される
    }
});

問題点

OAuth認証後、URLにcode/stateパラメータが含まれている場合、
woff.init()が自動的に認証を完了させます。
再度woff.login()を呼ぶ必要はありません。

デモ

URLパラメータの状態と認証フローを確認できます
